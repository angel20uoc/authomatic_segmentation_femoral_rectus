# authomatic_segmentation_femoral_rectus
Segmentación automática del músculo recto femoral en ecografía mediante deep learning (U-Net, Attention U-Net y U-Net++). Proyecto de TFM orientado a la extracción de parámetros morfométricos (área y grosor) y al análisis de sarcopenia en una cohorte multicéntrica de adultos mayores.
